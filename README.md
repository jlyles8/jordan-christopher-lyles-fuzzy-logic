# Medication Dosage Adjustment System Using Fuzzy Logic

**Author:** Jordan Christopher Lyles

This GitHub Pages project presents an academic prototype for a personalized medication dosage adjustment and medication allocation system using fuzzy logic. The project includes a professional webpage, expanded PDF report, full Python source code, sample output, original PowerPoint file, Wolfram Language companion file, and medical exclusion criteria CSV.

## Contents

- `index.html` - polished website report
- `code_appendix.html` - full source code and CSV appendix
- `docs/jordan_christopher_lyles_medication_fuzzy_logic_project.pdf` - professional PDF report
- `code/patient_profile.py` - synthetic patient profile generator
- `code/NUCL570_final.py` - fuzzy medication desirability model
- `data/medication_exclusion_criteria.csv` - prototype safety/exclusion criteria table
- `docs/sample_output.txt` - sample terminal output
- `docs/Final_NUCL_570_Presentation.pptx` - original project presentation
- `wolfram/fuzzy_logic_membership_model.wl` - Wolfram Language companion model

## Educational Disclaimer

This is an academic computational prototype. It is not a prescribing system, medical device, clinical decision support tool, or source of medical advice. Clinical use would require expert review, validation, regulatory assessment, and clinician oversight.
