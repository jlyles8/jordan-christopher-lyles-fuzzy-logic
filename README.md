# Medication Dosage Adjustment System Using Fuzzy Logic

Project by Jordan Christopher Lyles.

This GitHub Pages folder contains a responsive research webpage, full code appendix, patient-profile generator, medical exclusion criteria, PowerPoint source, and PDF report.

## GitHub Pages
Upload all files in this folder to a repository and enable GitHub Pages from the main branch root.

- `index.html` - polished responsive webpage
- `code_appendix.html` - full actual code and exclusion criteria without page overflow
- `code/NUCL570_final.py` - fuzzy logic model
- `code/patient_profile.py` - synthetic patient profile generator

- `assets/fuzzy_membership_functions.png` - membership function visualization
- `docs/Final_NUCL_570_Presentation.pptx` - original presentation
- `docs/jordan_christopher_lyles_medication_fuzzy_logic_project.pdf` - PDF report
